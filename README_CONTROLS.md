# Mind Map Controls Guide

Welcome to your professional Mind Mapping tool. This guide outlines all the available controls to help you build and organize your ideas efficiently.

## ⌨️ Keyboard Shortcuts

Speed up your workflow with these essential hotkeys:

| Key | Action |
|-----|--------|
| **Tab** | Add a **Child Node** to the selected node |
| **Enter** | Add a **Sibling Node** next to the selected node |
| **Delete** | Delete the selected node or drawing |
| **Ctrl + Z** | **Undo** last action |
| **Ctrl + Shift + Z** | **Redo** last action |

---

## 🖱️ Canvas Navigation

- **Pan**: Click and drag on the background (ensure the **Hand** tool is active).
- **Zoom**: Use your mouse wheel or the zoom controls in the bottom-left corner.
- **Select Node**: Click once on any node to select it and open the **Context Panel**.
- **Edit Text**: Click on any node label or drawing title/paragraph to start typing.

---

## 🎨 Drawing Tool (Regions)

Use the **Dotted Rectangle** icon in the right sidebar to mark conceptual regions:

1. **Draw**: Click and drag on the canvas background to create a dashed rectangle.
2. **Color Selection**: When the Draw tool is active, a side menu appears allowing you to switch between **Red** and **Green**.
3. **Auto-Switch**: After drawing a rectangle, the tool automatically switches back to **Panning** mode for convenience.
4. **Resizing**: Select a drawn rectangle in Panning mode to reveal resize handles on its edges and corners.
5. **Delete**: Hover over a drawing to reveal the **XL Delete Button (X)** in the top-right corner.

---

## 🛠️ Toolbar Features

### Header Menu
- **New Map**: Resets the canvas to a fresh start (requires confirmation).
- **Export**: Saves your current mind map (including drawings!) as a `.json` file.
- **Undo/Redo**: Revert or restore your changes.
- **Fit View**: Instantly centers and zooms the canvas to show your entire map.
- **Theme Toggle**: Switch between **Dark Mode** and **Light Mode**.

### Node Context Panel
When a node is selected, use the floating panel to:
- Change the **Node Shape** (Rounded, Pill, Square).
- Change the **Node Color** (Blue, Purple, Orange, etc.).
- Quickly add children or delete the node.

---

## 💾 Persistence
All your work—including nodes, custom positions, background drawings, and even text edits—is **automatically saved** to your browser's local storage. You can safely refresh the page without losing progress.
